# dotfiles - Narigo

Setup for devcontainers and macOS.

## Install

**macOS:**

```bash
./install-macos.sh
```

**Devcontainer:**

```bash
./install-devcontainer.sh
```

Both scripts add shared git config (aliases, colors, etc.) via `[include]` — your personal `user.name`, `email`, and signing config stay untouched.
